# Kilp
[![Downloads](https://img.shields.io/npm/dt/kilp.svg)](https://www.npmjs.com/package/kilp)

Kill process listening on port. **Only tested in mac.**

# Usage
CLI only.

## Global
##### Installation
    $ npm i kilp -g
##### Usage
    $ kilp <port>
##### Examples
    $ kilp 8080

## NPX
##### Direct Usage
    $ npx kilp <port>
##### Examples
    $ npx kilp 8080
